═══════════════════════════════════════════════════════════════════════

✅ ÉTAPE 1 - COPIER LES FICHIERS (2 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

MÉTHODE AUTOMATIQUE (Recommandée):
  1. Double-cliquez sur: install_xampp.bat
  2. Répondez "O" pour confirmer
  3. Attendez "✓ Installation réussie!"

MÉTHODE MANUELLE:
  1. Ouvrez l'Explorateur (Win+E)
  2. Allez à: C:\xampp\htdocs\
  3. Collez le dossier Keyce-KIAI dedans
  4. Résultat doit être: C:\xampp\htdocs\Keyce-KIAI\

═══════════════════════════════════════════════════════════════════════

✅ ÉTAPE 2 - DÉMARRER XAMPP (2 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Ouvrez XAMPP Control Panel
   (Cherchez: XAMPP Control Panel sur le Bureau ou dans Programmes)

2. Cliquez "Start" pour:
   ☑ Apache
   ☑ MySQL

3. Attendez qu'ils deviennent VERTS ✓

⚠️  SI ERREUR:
   - Attendez 10 secondes
   - Essayez de redémarrer (Stop → Start)
   - Si toujours en erreur, contact support XAMPP

═══════════════════════════════════════════════════════════════════════

✅ ÉTAPE 3 - IMPORTER LA BASE DE DONNÉES (3 minutes)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. Ouvrez phpMyAdmin:
   http://localhost/phpmyadmin

2. Cliquez sur l'onglet "Importer" (en haut)

3. Cliquez "Choisir un fichier"

4. Naviguez vers:
   C:\xampp\htdocs\Keyce-KIAI\db\

5. Sélectionnez:
   membres_du_groupe.sql

6. Cliquez "Exécuter" (en bas)

7. Attendez: ✓ "Importation réussie!"

✓ VÉRIFICATION:
   - À gauche, cherchez base: "KeyceKIAI"
   - Cliquez dessus
   - Vous devez voir table: "membres_du_groupe"

═══════════════════════════════════════════════════════════════════════

✅ ÉTAPE 4 - TESTER LA CONFIGURATION (1 minute)
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Allez à l'URL:
  http://localhost/Keyce-KIAI/diagnostic_xampp.php

Vérifiez que vous voyez:
  ✓ PHP Fonctionne
  ✓ Connexion à MySQL Réussie
  ✓ Table "membres_du_groupe" Trouvée
  ✓ Tout est Prêt!

❌ SI UNE ERREUR S'AFFICHE:
   - Lisez le message d'erreur
   - Relisez les étapes ci-dessus
   - Vérifiez que vous avez bien importé le SQL
